## 마크다운 기본문법

#### 1. 제목(header)

```
# h1
## h2
### h3
#### h4
##### h5
```
* 위와 같이 #의 개수로 header의 크기를 조절할 수 있다.

#### 2. 강조(emphasis)
```
이텔릭체는 *별표(asterisks)* 혹은 _언더바(underscore)_를 사용하세요.
두껍게 **별표(asterisks)** 혹은 __언더바(underscore)__를 사용하세요.
**_이텔릭체_와 두껍게**를 같이 사용할 수 있습니다.
취소선은 ~~물결표시(title)~~를 사용하세요.
```
